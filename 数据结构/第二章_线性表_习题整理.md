**二、算法设计题（仅第 6 题）**

**题目 6**

设计一个算法，通过一趟遍历确定长度为$$n$$的单链表中值最大的结点。

**思路**

链表带有头结点，用指针$$p$$遍历链表，$$maxp$$保存当前最大值结点；

初始化：$$maxp$$指向第一个有效结点，$$p$$从第二个结点开始遍历；

循环逐个比较，若$$p$$的数据大于$$maxp$$的数据，则更新$$maxp = p$$；

遍历结束后，$$maxp$$即为值最大的结点。

**C 语言完整代码实现**

  --------------------------------------------------------------
  c\
  #include \<stdio.h\>\
  #include \<stdlib.h\>\
  \
  // 定义单链表结点结构\
  typedef struct LNode {\
  int data;\
  struct LNode \*next;\
  } LNode, \*LinkList;\
  \
  // 创建带头结点单链表（尾插）\
  void CreateList(LinkList \*L, int arr\[\], int n) {\
  \*L = (LinkList)malloc(sizeof(LNode));\
  (\*L)-\>next = NULL;\
  LNode \*tail = \*L;\
  for (int i = 0; i \< n; i++) {\
  LNode \*p = (LNode\*)malloc(sizeof(LNode));\
  p-\>data = arr\[i\];\
  p-\>next = NULL;\
  tail-\>next = p;\
  tail = p;\
  }\
  }\
  \
  // 一趟遍历查找值最大的结点，返回结点指针\
  LNode\* FindMaxNode(LinkList L) {\
  if (L-\>next == NULL) { // 空链表\
  return NULL;\
  }\
  LNode \*maxp = L-\>next; // 初始最大值结点为第一个元素\
  LNode \*p = maxp-\>next;\
  // 一趟遍历所有剩余结点\
  while (p != NULL) {\
  if (p-\>data \> maxp-\>data) {\
  maxp = p;\
  }\
  p = p-\>next;\
  }\
  return maxp;\
  }\
  \
  int main() {\
  int arr\[\] = {23, 11, 45, 8, 67, 32};\
  int n = sizeof(arr) / sizeof(arr\[0\]);\
  LinkList L;\
  CreateList(&L, arr, n);\
  \
  LNode \*maxNode = FindMaxNode(L);\
  if (maxNode != NULL) {\
  printf(\"链表最大值结点的数据：%d\\n\", maxNode-\>data);\
  } else {\
  printf(\"链表为空！\\n\");\
  }\
  return 0;\
  }

  --------------------------------------------------------------

**算法说明**

**时间复杂度**：$$O(n)$$，仅遍历链表 1 次，$$n$$为结点个数；

**空间复杂度**：$$O(1)$$，仅使用常数临时指针，无额外数组 / 链表开辟；

边界处理：包含空链表判断，避免空指针访问崩溃。
